# claude_skills
Claude skills I've created and up for sharing.

## how to install
If you want the skill to be used across all your projects, copy it into `~/.claude/skills` (you might need to create the skills folder).

If you want the skill to be used in a single project, copy it into the `.claude/skills` of your project.

## Skills
`product-design-portfolio` - a skill that help you create your own portfolio website (html/css/js)
